# Numpy


## 1. 기초 개념
* Numpy에서는 모든 배열 값이 같은 타입이여야 하며, 각 차원을 축이라고 표현한다.
```
1차원 축(행) : axis(축) 0 -> Vector
1차원 축(행) : axis(축) 0 -> Matrixw
1차원 축(행) : axis(축) 0 -> Vector
```
